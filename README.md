# Statwing

Ruby wrapper for the Statwing Api

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'statwing'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install statwing

## Configuration

```ruby
Statwing.configure do |config|
	config.api_key = 'asdf'
end
```
